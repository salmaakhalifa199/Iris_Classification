# Iris Classification Project

This project classifies Iris flowers using K-Nearest Neighbors (KNN) and performs detailed visualization and evaluation.

## Features
- Dataset exploration and visualization
- Feature scaling
- K-Nearest Neighbors classification
- Cross-validation evaluation
- Confusion matrix visualization
- Mapping numerical predictions to species names

## Files
- `Iris_Classification.ipynb`: Main Jupyter notebook
- `IRIS.csv`: Iris dataset

## Technologies Used:
- Python
- Pandas, NumPy
- Scikit-Learn
- Matplotlib, Seaborn
